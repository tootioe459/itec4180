# Simple PHP Chat

A clean PHP + MySQL public chat room with session-based usernames and JSON API endpoints.

## Features

- Join with a username
- Shared public chat room
- Auto-refresh messages every 2 seconds
- PDO database access
- JSON API endpoints
- GitHub-ready structure
- Environment-based config
